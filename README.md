# 🍬 CINAMON

> *Catch them all in this retro-style creature collecting RPG!*

A Pokemon-inspired HTML5 adventure built entirely in a single file — no dependencies, no build tools, just pure vanilla JavaScript and Canvas. Explore forests, battle trainers, catch creatures, and conquer 3 gym leaders to become the ultimate Cinamon Champion!

---

## 🎮 Play Now

Simply open `cinamon/index.html` in any modern browser. That's it. No server required.

```bash
git clone https://github.com/lw-xiong/cinamon.git
cd cinamon
open cinamon.html
```

## ✨ Features

### 🌍 Open World Exploration
- **5 unique areas** to discover: Cinnabar Town, Route 1, Route 2, Mystic Forest, and Rocky Mountain
- Each area has its own terrain, wild creature encounters, and secrets
- Seamless map transitions with smooth fade effects

### ⚔️ Turn-Based Battle System
- Classic Pokemon-style turn-based combat with type effectiveness
- **10 creature types**: Normal, Fire, Water, Grass, Electric, Ice, Rock, Dark, Flying, Bug
- Strategic move selection with accuracy, critical hits, and status effects
- Level up your creatures, learn new moves, and watch them evolve!

### 🏟️ 3 Gym Challenges
- **Bloom Gym** — Flora's Grass-type garden gauntlet
- **Blaze Gym** — Ash's fiery forge of battle
- **Torrent Gym** — Marina's ocean of waves

Each gym features 2 trainers + the Gym Leader. Defeat them all to earn badges and prove your worth!

### 🎵 Original Chiptune Soundtrack
- 6 hand-crafted tracks: Title, Town, Route, Battle, Gym Battle, and Victory
- 8 sound effects for menu navigation, battles, catches, and more
- Dynamic music that changes based on your location and situation
- Press **M** to mute/unmute

### 📱 Full Mobile Support
- Touch-friendly D-pad and action buttons
- Responsive canvas scaling for any screen size
- Works on phones, tablets, and desktops

### 💾 Save System
- Full save/load via localStorage
- Continues where you left off: position, team, badges, money, gym progress
- Title screen with **Continue** or **New Game** options

## 🎮 Controls

### Keyboard
| Key | Action |
|-----|--------|
| `WASD` / `Arrow Keys` | Move |
| `Z` / `Enter` / `Space` | Action / Confirm |
| `X` / `Escape` | Menu / Cancel |
| `M` | Mute/Unmute Music |

### Mobile
- **D-Pad** (left side): Movement
- **A Button** (right side, red): Action / Confirm
- **B Button** (right side, blue): Menu / Cancel
- **START**: Open menu
- **SELECT**: Special action

## 🎯 Gameplay Tips

1. **Catch early, catch often** — Build a diverse team to handle different gym types
2. **Type matchups matter** — Water beats Fire, Fire beats Grass, Grass beats Water
3. **Visit the Nurse** — Heal your team at the Pokemon Center before tough battles
4. **Stock up on Cinaballs** — Buy from the Merchant in town ($100 for 5 balls + 3 potions)
5. **Explore everything** — Talk to NPCs, read signs, and discover hidden paths
6. **Save frequently** — Use the menu to save your progress
